## 알고리즘 스터디 순서
```
1. 언어별 기본 문법 공부
2. 코드업, 백준 기본 100제 문제풀이
3. 그리디 -> 완전 탐색(BFS, DFS) -> 기본 DP -> 그래프 이론 -> 심화 DP 및 문자열 순서대로 각 50 문제씩 풀이
4. 알고리즘 대회 / 기업 코딩테스트 기출문제 풀이
```
